CODEBREAKER - a number puzzle game

You have 5 attempts to guess a random 5-digit code. Each number is only
used once. Can you guess the code in time??

MIT License


TO BUILD:

Run BUILD.BAT to compile the program.
(Uses OpenWatcom to compile. You need to have your environment set up
correctly for this to compile.)

Run CLEAN.BAT to clean up after the build.
(This deletes the ERR, OBJ, and EXE files.)
